Selecione um arquivo ZIP contendo as imagens.
Instruções de Upload
Formatos Suportados:
  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • WebP (.webp)
Limitações:
  • Tamanho máximo: 16MB por arquivo
  • Resolução recomendada: até 4K
  • Upload em lote: até 50 imagens por ZIP
Dicas para PythonAnywhere:
  1. Upload em Lote: Use a funcionalidade ZIP para enviar várias imagens de uma vez
  2. Organização: Organize suas imagens em pastas por categoria antes de compactar
  3. Nomenclatura: Use nomes descritivos para facilitar a identificação
  4. Otimização: Redimensione as imagens antes do upload para economizar espaço
