## Especificações Técnicas

### Formato e Qualidade
- Formato: JPG para fotos, PNG para logos
- Qualidade: Alta resolução para web
- Otimização: Comprimidas para web (balanço qualidade/tamanho)
- Responsive: Imagens que funcionem bem em diferentes tamanhos

### Direitos Autorais
- Todas as imagens devem ter autorização para uso
- Preferir imagens próprias do projeto
- Respeitar direitos autorais de terceiros
- Incluir créditos quando necessário

### Acessibilidade
- Todas as imagens devem ter atributo alt descritivo
- Imagens decorativas devem ter alt=""
- Imagens informativas devem ter descrição clara

## Notas Importantes

1. **Lei Rouanet**: As logomarcas governamentais são obrigatórias e devem estar sempre visíveis
2. **Acessibilidade**: Todas as imagens devem ter texto alternativo apropriado
3. **Performance**: Imagens devem ser otimizadas para carregamento rápido
4. **Responsividade**: Devem funcionar bem em dispositivos móveis
5. **Qualidade**: Manter alta qualidade visual para transmitir profissionalismo
